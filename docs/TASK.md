# Interview Code Exercise

## Provision IAM \- Java Web Developer

Develop a Java Webapp to display information about past and future SpaceX launches and SpaceX rockets.

## Requirements

### General

* Develop a Java Webapp
* Utilize a Java framework such as Spring or similar
* Consume the SpaceX RESTful API within the Java Web Application and return results to be displayed in the browser
* Style and display information in an effective manner. Use the CSS framework of your choice such as Bootstrap.

### SpaceX Rockets

#### Using the /rockets SpaceX API endpoint:

* Retrieve and display rocket information on the front-end
* SpaceX API is consumed in the Java Webapp
* Include sort functionality within the Java Webapp, allowing the user to manipulate the sort field and direction from the front-end interface

#### Using the /launches API endpoint:

* For each Rocket, display a list of launches associated with that rocket â— List of launches is specific to a particular rocket
* SpaceX API is consumed in the Java Webapp
